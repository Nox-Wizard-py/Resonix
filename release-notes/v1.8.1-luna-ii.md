# 🌕 Resonix v1.8.1 — Luna-II

Luna-II is one of the biggest updates Resonix has ever received.  
This release introduces real-time shared listening, Shazam-style music recognition, AI-powered lyrics translation, massive UI redesigns, synchronization engineering upgrades, and deep performance optimizations across the entire app.

---

# 🎤 Shazam-Style Music Recognition

A complete music recognition system has now been integrated directly into Resonix.

### Added
- Full Shazam-inspired music recognition pipeline
- Dedicated `shazamkit` module and backend recognition routes
- Live microphone audio processing & decoding pipeline
- On-device audio resampling and WAV utilities
- Real-time animated particle sphere listening interface
- Interactive recognition bottom sheet with seamless song actions
- Direct navigation into search, player, and lyrics
- Recognition history screen with delete support
- Persistent foreground recording service for stable recognition sessions
- Quick-access microphone FAB above shuffle controls

### Improved
- Faster recognition flow
- Cleaner transitions between recognition and playback
- Better microphone lifecycle handling

# 🎧 Resonance — Listen Together

Introducing **Resonance**, Resonix’s brand-new real-time synchronized listening experience.

### Core Features
- Full WebSocket-powered sync architecture
- Dedicated `resonix-sync` backend system
- Real-time room-based shared listening
- QR-based room joining flow
- Automatic host detection with Crown indicators
- Seamless host transfer and recovery system

### BeatSync Synchronization Engine
Resonance now includes a custom-built synchronization protocol engineered for millisecond-accurate playback across devices.

### Added
- NTP-style clock synchronization
- RTT latency calculations
- Drift correction and compensation systems
- Playback timestamp alignment
- Audio preload gating system
- `AudioLoaded` and `LoadAudio` synchronization events
- SyncSnapshot support for mid-session joins
- Queue and timestamp recovery for new participants

### Connection Resilience
- Added stale-session recovery logic
- 30-second host grace window
- Improved reconnection handling
- Better background session stability

### Guest Restrictions
- Locked shuffle and loop controls for guests
- Improved room authority handling
- Safer playback ownership logic
